# Changelog

## [0.1.0] - 2025-11-05

### Added
- Initial beta release of obseditor script
- Support for opening external markdown files in Obsidian
- Automatic symlink creation for linked resources (images, PDFs)
- Recreation of original folder structure in Obsidian vault
- Multiple fi-le selection support from file explorer (Nemo and Nautilus)
- Also runs from the terminal
- Create new files with -n/--new flag
- Templater template `Sync Attachments.md` to "translate" additions, modifications, insertions of objects in Obsidian or an external markdown editor (Typora)
- Integration with Typora image handling
